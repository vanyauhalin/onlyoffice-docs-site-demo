Unsubscribe from the event

DELETE /api/2.0/CalendarApi/events/{eventId}/unsubscribe

Description

Unsubscribes the current user from the event with the ID specified in the request.

Request

Path Parameters

eventId object
Properties of eventId
unknown unknown

Examples

DELETE /api/2.0/CalendarApi/events/{eventId}/unsubscribe HTTP/1.1
Accept: application/json
Host: example.com
