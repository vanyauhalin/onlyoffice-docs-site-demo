Get version history

GET /api/2.0/files/file/{fileId}/edit/history

Description

Returns the version history of a file with the ID specified in the request.

Examples

GET /api/2.0/files/file/{fileId}/edit/history HTTP/1.1
Accept: application/json
Host: example.com

curl -L
  -X "GET"
  -H "Accept: application/json"
  "https://example.com/api/2.0/files/file/{fileId}/edit/history"
